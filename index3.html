Чему я научился за этот учебный год

Меня зовут Дамир, мне одиннадцать лет, я учусь в группе C2212. В этом учебном году я узнал много нового и интересного. Мне понравилось обучатся в TOP. Я изучил такие программы как: MitApp Inventor, Roblox Studio, YouTube, Программирование на Python, codepen.

MitApp Inventor

MitApp Inventor - это программа в которой можно создать приложения такие, как: пианино, лаберинт, канкулятор, палитру цветов, футболл и деже получилось сделать свой браузер. Но чтобы протестировать своё приложение, нужно подключится через телефон. MitApp Inventor разделён на две части - дизайн приложение и программа для приложения.

Roblox Studio

Roblox- популярная игра, а Roblox Studio- это программа для создания игр в Roblox. В Roblox Studio можно делать игры такие, как: паркур, лаберинт, гонки и даже хорор в котором нужно збегать от маньяка.

YouTube

В блоке YouTube я научился делать треллеры фильмов, монтировать видео и музыку,создание видео-двойника.

Программирование на Python

В Python я научился делать: игру крестики-нолики, электронный дневник...Так же в этой программе есть искуственный интелект. Python разделён на две части - первая часть в которой ты пишешь код для программы и вторая для тестирования программы.

Codepen

В блоке codepen я научился делать: меню, дизайн ,карусель, коментарии сайтов, добать картинки, cсылки, видео для сайтов.Codepen разделён на три части - HTML , CSS и JS
